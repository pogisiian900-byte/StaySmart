## Troubleshooting

**Can't find "Add Funds" button?**
- Make sure you're in Sandbox mode (not Live)
- Try logging in directly to sandbox PayPal
- Some accounts may need to be activated first

**Funds not showing?**
- Refresh the page
- Check you're looking at the correct account (`lhanie@business.example.com`)
- Verify account ID matches: `88UKGRJYCMYBN`

**Account not found?**
- Verify the account exists in your PayPal Developer Dashboard
- Check you're in Sandbox mode
- Try creating a new sandbox business account if needed
